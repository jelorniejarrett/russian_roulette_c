# Russian Roulette Simulation (C)

A Russian Roulette game simulation in command-line using C language where the player goes up against a simple AI opponent in a turn-based way.

## Features
- Random placement of bullets in the revolver's chambers
- Turn-based game for player and AI
- Chance of firing or passing up
- Random choices of opponent
- Ends after shooting down a player

## Coding Languages Used
- C (Standard Library)

## Explanation
- Place three bullets randomly into 7 chambers of the revolver
- Give players the option to fire or pass their turn
- If there is a bullet, lose the game
- Continue until a player gets shot

## Concepts Shown
- Arrays
- Loops and conditionals
- Random number generation
- State control in games
